Introdução CSS

o que é CSS

Css, ( Cascading Style Sheets oi folhas de estilos em cascatas ) é uma linguagem de estilo utilizada para descrever a apresentacao de domumentos escritos em html ou xml. ela define a aparencia visual de uma pagina web, controlando elementos como cores, fontes, espacamentos, layout e desing

existem 3 maneiras de aplicar CSS

Inline

Na mesma linha, atraves de um atributo chamado style

interno

dentro de um mesmo documento HTML, atraves de uma tag chamada Style

externo

crianda um novo arquivo .css e utilizando seletores
